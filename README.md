# ⚽ Süper Lig Standings API

Next.js ile geliştirilmiş dinamik Süper Lig puan durumu uygulaması.

## 🚀 Özellikler

- TFF verilerini API üzerinden çekme
- JSON API sistemi
- Responsive puan durumu widgetı
- Takım logoları
- Tailwind CSS tasarımı
- Next.js App Router yapısı

## 🛠 Teknolojiler

- Next.js
- React
- Tailwind CSS
- Node.js

## 📦 Kurulum

```bash
npm install
npm run dev
```

## 🔗 API Endpoint

```bash
/api/standings
```

## 📷 Özellikler

- Şampiyonlar Ligi renkleri
- Avrupa kupaları göstergeleri
- Düşme hattı
- Responsive sidebar widget yapısı
